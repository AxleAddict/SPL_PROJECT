This is a Special Intern Projec.
This project is based on MERN stack

## Local Setup

### Requirements:

MongoDB
NodeJS

Run 'npm install' to get all the required dependencies.<br />
<b>NOTE:</b> The appplication uses Cisco UI, hence it is necessary to connect to the Cisco VPN to get all the dependencies

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the UI in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `node server/server.js`

Launches the backend server at http://localhost:8080.<br />
<b>NOTE:</b> mongoDB server should be running in the background for this to work
